# Responsive Landing Page

## Overview
This project is a responsive landing page designed for a fictional startup or product. The webpage is built using HTML and CSS with modern layout techniques such as Flexbox and CSS Grid.

## Features
* Responsive design
* Hero section
* Features section
* Testimonials section
* Footer section
* Mobile-friendly layout

## Technologies Used
* HTML5
* CSS3
* Flexbox
* CSS Grid

## Purpose
To practice responsive web design and modern frontend layout techniques.
